# 微信小程序实现电商信息交流平台
## 安装项目
```
1.git clone https://github.com/gwiguo/commerce-platform  
2.npm install
3.微信开发者工具->工具->构建npm
```
## 项目说明：
该项目属于小程序期末作业，所以数据都是本地模拟的，在mock文件里下
